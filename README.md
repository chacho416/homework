# ⚡ Chachos Shonens

¡Bienvenido a **Chachos Shonens**! Este es un catálogo interactivo de manga diseñado para ofrecer una experiencia de usuario rápida, minimalista y con un estilo visual de alto contraste. 

Este proyecto fue desarrollado como una práctica de ingeniería de software, enfocada en el consumo de APIs externas y la creación de interfaces dinámicas con **Next.js**.

---

## 🚀 Características

* **Buscador en Tiempo Real:** Encuentra cualquier manga disponible en la base de datos de MyAnimeList.
* **Modo Oscuro "Pure Black":** Diseño optimizado para pantallas OLED con un fondo negro absoluto y acentos verdes.
* **Diseño Responsivo:** Totalmente adaptado para dispositivos móviles y escritorio.
* **Detalles Completos:** Información sobre sinopsis, volúmenes, estado de publicación y calificaciones.
* **Tecnología Server-Side:** Renderizado eficiente utilizando las últimas funciones de Next.js.

---

## 🛠️ Tecnologías Utilizadas

* **Framework:** [Next.js 13+](https://nextjs.org/) (App Router)
* **Librería UI:** [React](https://reactjs.org/)
* **Estilos:** [Tailwind CSS](https://tailwindcss.com/)
* **Iconos:** [React Icons](https://react-icons.github.io/react-icons/)
* **API:** [Jikan API](https://jikan.moe/) (MyAnimeList)
* **Despliegue:** [Vercel](https://vercel.com/)

## 🔧 Instalación y Configuración

Si quieres ejecutar este proyecto localmente, sigue estos pasos:

1.  **Clonar el repositorio:**
    ```bash
    git clone [https://github.com/chacho416/homework.git](https://github.com/chacho416/homework.git)
    ```
2.  **Instalar dependencias:**
    ```bash
    npm install
    ```
3.  **Iniciar el servidor de desarrollo:**
    ```bash
    npm run dev
    ```
